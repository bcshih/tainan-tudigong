🔮 專案願景

What is this project about?

本專案將台灣傳統民間信仰的神明分工體系映射到現代 AI Agent 架構上。 最高層的土地公（Tudigong）扮演社區總管，統籌調度各層級的 Agent， 就像真正的土地公守護一方水土一樣，只不過祂的「神力」來自 Google Gemini LLM、 IoT 感測器與社群媒體數據。

系統使用 Contract Net Protocol（合約網協議） 讓 Agent 之間動態協商任務分配—— 土地公發出任務提案，下層 Agent 競標，最終擇優發包。這套機制讓系統具備高度彈性與自組織能力。 前端則透過獨創的 A2UI（Agent-to-User Interface）協議， 讓 Agent 能主動推送結構化的 UI 元件（調查卡片、地圖疊層、許願介面等）給使用者。

資料層運用 Warm Data 概念——不只記錄冰冷事實，更記錄「關係」： 誰跟誰是鄰居、哪條街跟哪間廟有歷史關聯、哪個議題牽動哪些利害關係人。 這讓 AI 的判斷不只是數據驅動，更是脈絡感知的。

📜 核心協議

Protocols — 驅動系統運作的三大協議

📋 Contract Net Protocol

deg/schemas/contracts.py

Agent 之間的動態任務協商機制。 土地公發出 ProposalMsg（任務提案）， 下層 Agent 回覆 BidMsg（競標）， 土地公評估後發出 AwardMsg（得標通知）。 狀態流轉：PROPOSED → BID → AWARDED / REJECTED → DONE。

🤝 A2A (Agent-to-Agent)

a2a-sdk · FastAPI JSON-RPC

Agent 間的通訊標準。 每個下層 Agent（地基主、巡境使、戶爺）都運行自己的 A2A Server， 透過 FastAPI 暴露 JSON-RPC 端點，讓土地公能遠端派發任務、接收結果。 基於 a2a-sdk 實作。

🖥️ A2UI (Agent-to-User Interface)

deg/a2ui/ · v0.3

獨創的Agent→使用者介面協議。 Agent 不只回傳文字，更透過 A2UIBuilder 構建結構化 UI 元件（SurveyCard、MapOverlay、WishSurface、ClarifyGroup）， 以 JSON 推送至前端自動渲染。讓 Agent 的回應可視化、可互動。

🗄️ MCP (Model Context Protocol)

deg/mcp/spatial_db/

透過 MCP 協議暴露空間資料庫能力。 讓 LLM Agent 能以標準化方式查詢地理空間資訊—— 哪條街在哪裡、周邊有什麼設施、距離多遠，為 Agent 的在地決策提供地理脈絡。

🌊 Warm Data

deg/warmdata/store.py · SQLite

受 Nora Bateson「Warm Data」概念啟發的關係資料儲存。 以 Subject-Predicate-Object 三元組記錄實體間的多維度關係（鄰里、歷史、利害關係）， 讓 AI 不只看到數據點，更能感知脈絡與關聯。

🔌 Data Adapters

deg/adapters/

資料接口層——SensorAdapter 接入 IoT 感測器數據 （溫度、濕度、PM2.5、噪音），SocialAdapter 接入社群媒體貼文 （居民投訴、好評等）。目前使用 seed JSON 模擬，未來可接入真實來源。

💡 設計洞察

Why this architecture is interesting

🔹 文化隱喻即架構 — 不是刻意套用民間信仰的名字做包裝，而是真正利用了神明體系的「分工邏輯」： 土地公管一方（Orchestrator）、地基主守一屋（Building Inspector）、 巡境使走四方（Patrol）、戶爺管一戶（Household）、五營調兵將（Emergency Dispatch）。 這套分層恰好映射到真實社區治理的粒度需求。

🔹 Contract Net 賦予彈性 — Agent 之間不是靜態的指令鏈，而是動態的「招標-投標-得標」過程。 這意味著系統可以根據當下各 Agent 的能力與負載，自組織地分配任務。

🔹 A2UI 突破 chatbot 限制 — 傳統 chatbot 只能回傳文字。A2UI 讓 Agent 回傳「結構化 UI」—— 卡片、地圖、互動問答——讓使用者體驗從「讀文字」升級為「看 Dashboard」。

🔹 Warm Data 讓 AI 有溫度 — 不只記錄「國華街的 PM2.5 是 35」，更記錄「國華街跟大天后宮有三百年的歷史淵源」、 「賣虱目魚粥的阿婆跟隔壁修鞋師傅是四十年的鄰居」。 這些關係讓 AI 的判斷具備人文脈絡。
